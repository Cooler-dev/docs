# 后端开发文档

## HTTP API

无特殊说明时 API 调用响应格式

| 字段名 | 数据类型 | 默认值 | 说明         |
| ------ | -------- | ------ | ------------ |
| ok     | Boolean  | true   | 接口调用结果 |
| msg    | String   | ~      | 返回信息     |



### 获取基本信息

#### GET`/api/`

参数: 无

响应:
| 字段名  | 数据类型 | 默认 | 说明               |
| ------- | -------- | ---- | ------------------ |
| version | String   | ~    | 后端版本号         |
| db      | Boolean  | true | 数据库配置是否正确 |

### db

> 数据库引擎类

#### db.init

此方法用于根据配置文件初始化数据库

参数: 无

返回: 无

#### db.put

此方法用于向数据库内添加键值

参数: 

| 参数  | 类型   | 说明   |
| ----- | ------ | ------ |
| key   | String | 关键字 |
| value | Any    | 值     |
